# ALM Toolkit

Outils ALM/Risk interactifs pour risk managers et professionnels bancaires.

Chaque outil est un **fichier HTML unique** — zéro dépendance, zéro backend, fonctionne offline. Ouvrez dans n'importe quel navigateur.

## Outils disponibles

| Vol. | Outil | Concept | Lien |
|------|-------|---------|------|
| 1 | **EVE Sensitivity Calculator** | Sensibilité ΔEVE aux chocs de taux (±100/±200 bps) par bucket de maturité | [Ouvrir](https://suiteforgeops.github.io/alm-toolkit/EVE-sensitivity.html) |

## Comment utiliser

1. Cliquez sur le lien ci-dessus — l'outil s'ouvre dans votre navigateur
2. Modifiez les montants Assets/Liabilities par bucket
3. Sélectionnez un scénario de choc
4. Le waterfall et les KPIs se recalculent en temps réel

Pour une utilisation offline : clic droit → "Enregistrer sous" → ouvrez le fichier HTML depuis votre machine.

## À propos

Ces outils sont **pédagogiques**. Ils illustrent les concepts réglementaires (IRRBB, Bâle III, EBA) de manière interactive mais ne remplacent pas les systèmes de production bancaires.

Série publiée par [Nicolas Dojcilovic](https://www.linkedin.com/in/nicolasdojcilovic/) — consultant senior ALM/Risk avec 20+ ans d'expérience (BNP Paribas, Société Générale, Ayvens).

Chaque outil est construit en 30 minutes avec [Claude](https://claude.ai).
